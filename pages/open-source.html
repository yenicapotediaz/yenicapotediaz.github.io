Kraken

Kraken is the command-line interface for Kraken-lib.

Kraken-lib

Kraken-lib is an orchestration and cluster level management system for Kubernetes. It uses a yaml configuration file for all aspects of the both the Kubernetes cluster and the infrastructure that is running it.

Kraken-tools

The base layer for the Kraken project.

Kraken-charts

Helm charts repo for deploying a Kraken cluster with the additional functionality an application may require.
